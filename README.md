# 🍔 Food Delivery Website

A modern food delivery web application built using HTML, CSS, and JavaScript.

## 🚀 Features

* Responsive design (mobile + desktop)
* Smooth navigation with fixed navbar
* Product listing with dynamic cart
* Add/remove items with quantity control
* Authentication (Login/Logout using localStorage)
* Protected cart (login required)
* Order button with smart navigation

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript
