# 🍔 Food Delivery Website

A modern and responsive **Food Delivery Web Application** built using HTML, CSS, and JavaScript. This project allows users to browse food items, add them to the cart, and manage their orders interactively.

## 📌 Features

* 🛒 Add items to cart
* ➕ Increase / ➖ Decrease item quantity
* 🧾 Dynamic cart updates
* 📦 Product data loaded from JSON
* 💻 Responsive design (works on mobile & desktop)
* ⚡ Fast and interactive UI

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & Layout
* **JavaScript (ES6)** – Functionality & DOM Manipulation
* **JSON** – Product data handling
